# B Frontend

## Description

The frontend interface for the B AI-powered development platform. Built with React, Vite, and Tailwind CSS, providing an intuitive user interface for interacting with the B system's backend services.

## Features

- Integrated workspace for code development
- Chat interface for AI interactions
- Project management tools
- Agent status dashboard
- GitHub integration
- Dark mode support

## Technology Stack

- **Frontend Framework**: React
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **State Management**: React Context
- **WebSocket**: Real-time communication

## Setup Instructions

1. **Prerequisites**:

   - Node.js (version 16 or higher)
   - npm or yarn

2. **Clone the Repository**:

   ```bash
   git clone <repository-url>
   cd frontend
   ```

3. **Install Dependencies**:

   ```bash
   npm install
   ```

4. **Environment Configuration**:

   - Copy `.env.example` to `.env` and configure your environment variables

5. **Run Development Server**:

   ```bash
   npm run dev
   ```

6. **Build for Production**:
   ```bash
   npm run build
   ```

## Development

- Use `npm run preview` to preview the production build
